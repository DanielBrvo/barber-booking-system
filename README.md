# 💈 Booking System - Barbershop
Front-end barber booking system for managing appointments and schedules.

## 🚀 Objective
    Create a system where clients can:
    - View available times
    - Book an appointment
    - Simulate a basic appointment calendar

## 🛠️ Technologies Used
    - HTML
    - CSS
    - JavaScript

## 📂 Project Structure
    - index.html → Homepage
    - styles.css → Styles
    - script.js → Booking Logic
