# 💈 Booking System - Barbershop

A front-end booking system for a barbershop that allows users to view available time slots and schedule appointments.

## 🚀 Features
- View available appointment times
- Book appointments interactively
- Simulated scheduling system
- Dynamic UI updates with JavaScript

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## ▶️ How to Use
1. Open `index.html` in your browser
2. Select an available time
3. Book an appointment

## 📂 Project Structure
- `index.html` → Main structure
- `styles.css` → Styling
- `script.js` → Booking logic
- `img/` → Image assets
